The greeting widget is made up of the main wrapper .greeting-widget as well as the head and body within this wrapper, .greeting-widget-header and .greeting-widget-body.

The content within each section can be styled with the text utilties for font size, weight, etc.

By default the weather, location and time only shows if the user allows the browser to capture their location.


<div class="greeting-widget">
    <div class="greeting-widget-header" hidden>
        <img src="" class="greeting-widget-header-icon" id="weather-icon">
        <div class="text-sm font-weight-bold text-uppercase" id="user_location"></div>
        <div class="text-xl font-weight-light" id="user_temperature"></div>
        <div class="text-sm">
            <span id="user_day"></span>,
            <span id="user_time"></span>
        </div>
    </div>
    <div class="greeting-widget-body">
        ...
    </div>
</div>
