This example uses the Content navigation to the left as reference.

Any content can become sticky, and in this case it is usually a card.

To make a card sticky add the class .sticky-nav to the .card. However, to have a smooth functioning sticky nav it is required to wrap the card in a <div> tag.

The sticky nav activates when the window makes contact with a trigger. The recommended trigger is a wrapper for the sticky nav using the class .sticky-nav-wrapper.


<div class="sticky-nav-wrapper">
    <div>
        <div class="card sticky-nav">
            ...
        </div>
    </div>
</div>
